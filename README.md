# SpringBoot个人博客项目
项目原型是b站李仁密老师的Spring Boot开发小而美的个人博客，持久层改用了mybatis。

博客地址：http://47.107.230.178:8081/      或者（2022/4/11之后可以访问）www.zouyt.xyz
## 项目的技术栈：
  - 前端UI：Semantic UI框架 + thymeleaf模板
  - 后端：Spring Boot + mybatis
  - 数据库：MySQL
 ## 项目环境
  - jdk1.8
  - Maven 3.8.1
  - MySQL 8.0.17
